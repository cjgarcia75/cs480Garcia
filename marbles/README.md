## Building
To build and run this program, navigate to the correct directory. Make a new 'build' directory, cmake in previous directory, then make in the build directory.

Example:
```
mkdir build
cd build
cmake ..
make
./marbles
```

## Controls
Move camera:

    w: up

    s: down
    
    a: left
    
    d: right
    
    up: zoom in
    
    down: zoom out
    
    left/right: change focus
